# 🚀 Pranshu K. // Premium Portfolio

A high-agency, systems-first portfolio landing page designed with **Next.js (App Router)** and **GSAP**. 

### 🏗️ Technical Architecture
- **Core**: Next.js 16 (Turbopack)
- **Styling**: Tailwind CSS / Vanilla CSS (Custom Design System)
- **Visuals**: Three.js / @react-three/fiber (WebGL Dot Matrix Background)
- **Animations**: GSAP (Smooth Stacking Parallax, MagicBento Grid, Full-Screen Reveal)
- **Performance**: Lenis (Physics-based Smooth Scrolling)

### 🏎️ Features
- **Cinematic Stacking Parallax**: Content layers slide vertically over a fixed Hero section with deep shadows.
- **MagicBento Grid**: Interactive skill-set architecture with spotlight tracking and magnetic particles.
- **Foundry-grade Projects Reveal**: Side-by-side row architecture for flagship products with high-fidelity asset frames.
- **Contextual Signals**: High-contrast typography and subtle inner glows focused on institutional credibility.

### 🛠️ Usage
1. Clone the repository
2. Install dependencies: `npm install`
3. Run dev server: `npm run dev`

### 🎨 Design Philosophy
Designed for **Total Ownership** and **Execution Depth** — emphasizing high-leverage products over side projects.
