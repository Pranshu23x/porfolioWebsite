# 🚀 Projects

## 🧭 Voyage (Flagship)

### What It Is
An **agentic AI platform** that automates job discovery, ranking, and resume optimization.

---

### Why It Exists
Job hunting is inefficient by design:
- Too much noise  
- Poor personalization  
- Manual, repetitive effort  

Voyage turns this into a **structured, automated pipeline.**

---

### What It Does
- Aggregates jobs across platforms (LinkedIn, Indeed, Naukri)  
- Dynamically ranks them based on candidate fit  
- Adapts resumes per role using AI  

This is not a “job board.”  
It’s a **decision + execution engine.**

---

### System Design Thinking
- Multi-source ingestion + normalization  
- Semantic parsing + embeddings  
- Ranking via similarity scoring + weighted heuristics  
- Resume optimization via LLM pipelines  
- Modular backend deployed on AWS for scalability and reliability  

---

### Why This Matters (Hiring Lens)
This project demonstrates:
- Ownership of a real-world problem  
- Ability to design multi-component systems  
- Practical application of AI beyond prompts  
- Experience deploying production systems on AWS  

---

### Status
Actively building and iterating.  
Focused on shipping a usable system, not a concept.

## 🧠 RippleCode

### What It Solves
LLMs struggle with large repositories due to context limits.

---

### Solution
A system that converts repositories into structured XML + optimized chunks, enabling better LLM comprehension.

---

### Key Outcomes
- ~38% token reduction vs JSON  
- Improved code understanding for large projects  
- 250+ developers using it  

---

### Engineering Takeaways
- Fixed memory bottlenecks using streaming + batching  
- Backend deployed on AWS handling real user traffic
